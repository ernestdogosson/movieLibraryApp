# Movie Library

A simple movie tracking application built with vanilla JavaScript and Tailwind CSS.

## Features

- Add, remove, and manage your movie collection
- Mark movies as watched/unwatched
- Filter movies by status (All, Watched, Unwatched)
- Real-time statistics dashboard
- Responsive design with Tailwind CSS

## Technologies Used

- HTML5
- Vanilla JavaScript (ES6+)
- Tailwind CSS (via CDN)
- No external dependencies or frameworks

## How to Use

1. Open `index.html` in your browser
2. Browse the pre-loaded movie collection
3. Click "+ Add Movie" to add new films
4. Use the filter buttons to view specific categories
5. Mark movies as watched or delete them as needed

## Live Demo

Deployed on GitHub Pages: [View Project](https://ernestdogosson.github.io/movieLibraryApp/)

## Challenges & Lessons Learned

**Challenges Tackled:**

- Managing state without a framework - learned to keep data and UI in sync
- Dynamic DOM manipulation - safely handling user input and escaping characters
- Responsive design patterns - creating a mobile-friendly interface
- Event delegation and cleanup - preventing memory leaks

**Key Takeaways:**

- Vanilla JavaScript can build complex applications without frameworks
- Tailwind CSS CDN simplifies development and deployment
- Component thinking helps organize even small projects
- User input validation is crucial for robust applications
